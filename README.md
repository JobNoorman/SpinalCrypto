# SpinalCrytpo

This repository contains cryptographic Spinal IPs. This work is currently in progress.  

## IP available

### Symmetric

- AESCore_Std (128/192/256-bit)
- DESCore_Std
- TripleDESCore_Std


### Hash

- MD5Core_Std


### MAC

- HMACCore_Std (tested with MD5)
